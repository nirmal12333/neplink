# Nepal-Themed Images for NepLink

This document describes the Nepal-themed images that have been created to enhance the visual appeal of the NepLink website.

## Image Categories

We have created the following images organized by category:

### Category Images (300x200 pixels)
- `news.png` - Blue theme for news section
- `marketplace.png` - Green theme for marketplace section
- `community.png` - Purple theme for community section
- `jobs.png` - Red theme for jobs section
- `freelance.png` - Orange theme for freelance section
- `rentals.png` - Teal theme for rentals section

### Feature Images (400x300 pixels)
- `feature-news.png` - Enhanced news feature image
- `feature-marketplace.png` - Enhanced marketplace feature image
- `feature-jobs.png` - Enhanced jobs feature image
- `feature-rentals.png` - Enhanced rentals feature image
- `feature-community.png` - Enhanced community feature image
- `feature-freelance.png` - Enhanced freelance feature image

### Hero Image (1200x400 pixels)
- `hero.png` - Main hero image with Nepal flag colors

## Usage

All images are stored in the `img/category/` directory and are automatically used throughout the website:

1. **Feature Cards** - Display category images in the main features section
2. **Sector Pages** - Each sector page uses its corresponding category image in the header
3. **Content Cards** - News, marketplace, jobs, and rental cards use category images as fallbacks
4. **Hero Section** - Main page hero section uses the hero image as background

## Design Elements

The images incorporate:
- Nepal flag colors (red, blue, white)
- Relevant icons for each category
- Subtle patterns and gradients
- Text labels for clarity
- Consistent styling across all images

## CSS Enhancements

The `styles/nepal-theme.css` file provides additional visual enhancements:
- Improved hover effects
- Better shadows and transitions
- Nepal color palette integration
- Responsive design improvements
- Enhanced button and card styling

These enhancements make the website more visually appealing and comfortable for users, with a consistent Nepal-themed aesthetic throughout.
